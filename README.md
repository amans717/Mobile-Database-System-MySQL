# 📱 Mobile Database System | MySQL

## 📌 Project Overview
This project involves designing and developing a **relational database management system** for storing and managing mobile phone information using **MySQL**. The database enables efficient storage, retrieval, and management of mobile specifications, pricing details, and related information through structured SQL operations.

---

## 🎯 Project Objective
The objective of this project is to design a normalized relational database that efficiently manages mobile data while ensuring data integrity, minimizing redundancy, and supporting efficient querying and reporting.

---

## 🛠️ Tools & Technologies
- MySQL
- SQL
- Relational Database Management System (RDBMS)
- ER Diagram
- Database Normalization

---

## 📂 Database Description
The database stores information related to mobile phones, including product specifications, brands, pricing, storage variants, and other relevant details using relational tables.

---

## 📈 Key Features

- Store and manage mobile specifications and pricing information.
- Perform Create, Read, Update, and Delete (CRUD) operations.
- Retrieve and filter records using SQL queries.
- Maintain relationships between multiple database tables.
- Generate reports using SQL queries.

---

## ⚙️ Project Workflow

1. Gathered system requirements and identified required entities.
2. Designed the Entity Relationship (ER) Diagram.
3. Created normalized database tables.
4. Established primary and foreign key relationships.
5. Implemented CRUD operations using SQL.
6. Wrote SQL queries for filtering, sorting, searching, and reporting.
7. Optimized the database structure to ensure consistency and reduce redundancy.

---

## 🔍 Key Features Implemented

- Designed a normalized relational database schema.
- Established table relationships using primary and foreign keys.
- Implemented SQL queries for data retrieval and reporting.
- Performed CRUD operations for efficient data management.
- Maintained data integrity through normalization and relational constraints.

---

## ✅ Database Design Benefits

The database provides an efficient and scalable solution for managing mobile product information while ensuring data consistency, faster query execution, and reduced redundancy. Its normalized structure supports easy maintenance and future expansion.

The system enables users to:
